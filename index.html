Vos données. Vos règles.

Trois niveaux d'IA selon vos exigences de confidentialité : du cloud standard au 100 % local sur vos propres ordinateurs ou serveurs.
Vous choisissez ce qui convient à votre activité.

Modèles IA 100 % locaux

Pour les cas les plus sensibles, nous déployons des modèles open-source (Mistral, Llama) directement sur vos ordinateurs, vos serveurs ou votre cloud privé. Aucune donnée ne quitte votre infrastructure.

  • Self-hosted sur votre matériel ou cloud privé
  • Fonctionnement même hors-ligne possible
  • Idéal pour santé, finance, juridique

Conformité nLPD & RGPD

Documentation contractuelle complète : accord de traitement des données (DPA), registre des sous-traitants et processus de gestion des droits utilisateurs.

  • DPA fourni pour chaque mission
  • Sous-traitants identifiés et documentés
  • Droits d'accès, rectification, effacement

Hébergement Suisse ou Europe

Vous choisissez la juridiction. Pour les workflows critiques, nous configurons l'infrastructure chez des fournisseurs suisses ou européens uniquement.

  • Infomaniak, Exoscale (Suisse)
  • OVH, Scaleway (France)
  • Compatibilité avec votre cloud existant

Vos outils, vos accès

Nous nous connectons à vos systèmes via APIs officielles et OAuth. Les clés d'accès restent dans votre coffre-fort. Vous gardez le contrôle, à tout moment.

  • Aucune création de compte parallèle
  • Révocation possible en un clic
  • Aucune copie hors de votre périmètre

Chiffrement standard

Toutes les communications et stockages temporaires sont chiffrés selon les standards de l'industrie. Aucun raccourci sur la sécurité technique.

  • TLS 1.3 pour toutes les communications
  • AES-256 pour les données au repos
  • Secrets isolés dans votre vault

Audit & traçabilité

Chaque action exécutée par un agent est journalisée. Vous pouvez consulter, auditer ou annuler n'importe quelle opération à tout moment.

  • Logs horodatés de toutes les actions
  • Tableau de bord d'audit en temps réel
  • Export pour vos contrôles internes
